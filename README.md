# Espace Élève

Espace Élève est un portail web simple destiné aux étudiants pour accéder à leurs ressources académiques, gérer leurs devoirs et consulter leurs notes.

## Fonctionnalités
- Tableau de bord étudiant
- Soumission de devoirs
- Consultation des notes
